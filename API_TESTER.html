SDK — Public API (ใช้ X-API-Key)
ทุก request ใช้ header X-API-Key — ใส่ค่าด้านบน
GET/v1/domainsรายการโดเมนที่ใช้ได้
POST/v1/mailbox/createสร้าง mailbox ชั่วคราว
GET/v1/mailbox/:idดูข้อมูล mailbox + message count
PATCH/v1/mailbox/:idต่ออายุ TTL / เปลี่ยนสถานะ
GET/v1/mailbox/:id/messagesดึงรายการเมล์ (inbox)
GET/v1/mailbox/countนับจำนวน mailbox ของ tenant
GET/v1/message/:idอ่านเมล์ฉบับเต็ม + ไฟล์แนบ
DELETE/v1/message/:idลบ message (ถาวร)
GET/v1/attachment/:idดาวน์โหลดไฟล์แนบ (R2 URL)
DELETE/v1/mailbox/:idลบ mailbox (soft delete)
